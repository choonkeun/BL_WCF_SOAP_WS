# BL_WCF_SOAP_WS
BasicHttpBinding and Use only one URL to POST and "SOAPAction" will be used for "Action" such as Read, Add, Update, Delete
HTTP Header requires "content-type: text/xml; charset=utf-8" and 
SOAPAction: http://tempuri.org/IService1/GetToDos or
SOAPAction: http://tempuri.org/IService1/AddOrUpdateToDo
